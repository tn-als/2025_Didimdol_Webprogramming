1. (인터넷   )은 전 세계를 연결하는 국제 정보 통신망으로, 컴퓨터나 스마트폰 같은 디지털 기기로 연결되어 사람들이 정보를 공유할 수 있는 공간입니다.

2. ( WC3 )는 HTML표준을 비롯한 웹 표준안을 제작하거나 제안하느 국제적인 웹 표준화 단계입니다.

3. ( 웹페이지 )은 웹 페이지의 서식이나 구조를 표현하는 정보입니다

4. ( java script )는 HTML 페이지에서 사용자 반응 등을 처리하는 스크립트 작성 언어로 표준 명칭은 ( ECMA script )입니다.

5. ( 크로스플랫폼 )은 한 번 개발해서 여러 장치(플랫폼)에 대응하는 것을 의미합니다.

6. (  주석)은 프로그램 실행에 영향을 미치지 않으면서 설명용으로 사용하는 코드를 의미합니다.
 
7. 외부 스타일시트를 HTML 페이지에 불러오려면 ( link )태그를 사용합니다.

8. ( h1 )태그가 가장 큰 제목 태그고, ( h6 )태그가 가장 작은 제목 태그입니다.

9. 목록은 ul 태그와 ol 태그 내부에 (li  )태그를 넣어서 만듭니다.

10. a 태그로 빈 링크를 만들 때, href 속성에 일반적으로 사용되는 기호는 ( "" )입니다.

11. ( post )방식은 GET 방식과 달리 비밀스럽게 데이터를 전달합니다.

12. ( input )태그는 목록으로 보여주는 항목 중 하나 또는 여러 개를 선택할 떄 사용하는 입력 양식 요소입니다.

13. div 태그는 공간을 ( 블록 )형식으로 분할합니다.
